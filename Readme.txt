This project consists of three files:
1. Index.html - Main web page.
2. style.css - Style sheet for html elements
3. script.js - Javascript file which has all the logic and makes appropriate API calls

Steps to execute:
1. Copy all the files to the same directly
2. Open index.html in a browser
3. Click on "Get Files" button to display list of files and their details (Name, size, Download link) from the pre-created Github repository
